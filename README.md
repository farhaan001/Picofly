# Picofly 🐦

**Tap. Flap. Don't Crash.** 

Picofly by AA is an addictive one-tap arcade game. Tap to flap your bird, dodge pipes, and chase a high score. Includes Day & Night backgrounds, smooth 60FPS gameplay, instant restart, and a lightweight 15MB APK. Simple to learn, hard to master. Get Ready! Tap to Start. Created by developer AA
